F.A.Q - Perguntas Frequentes

Pra que serve o Brunch?

O objetivo do Brunch é ajudar cada pessoa a achar o estabelecimento ideal para visitar através de um sistema básico e intuitivo.

 Como faço para criar minha conta?

Na tela inicial do sistema, existem três formas de se cadastrar. Se você já possuir uma conta no Facebook ou Google poderá utiliza-las. Caso não possua faça um cadastro simples diretamente no site.

Como faço para sair do sistema?

No menu de opções clique em Sair.

Como localizar um restaurante?

Após entrar com seu usuário e senha, clique na primeira barra de busca e insira uma palavra chave e o sistema apresentará as buscas.

Onde ficam as informações adicionais dos estabelecimentos?

Abaixo do mapa de pesquisa traz alguns dados adicionais. Caso o estabelecimento seja um parceiro, trará dados mais completos dele.

Como conveniar meu estabelecimento?

No menu superior esquerdo, nas opções escolha a opção Seja Um Parceiro. O sistema solicitará os dados do seu estabelecimento e de pagamento.
